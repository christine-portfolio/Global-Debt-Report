# Global-Debt-Report
This repo contains a beginner-friendly debt report project done in Powerbi.

## Tools Used
-Power Query Editor
-Powerbi

## Project Overview
This project analyzes the Global debt rate of 84 countries from 1950-2022 a time span of 72 years.
It explores the yearly shift in debt composition, countries with high debt growth and debt inflation indicator using interactive dashboards.

## What I Learned
- How to clean messy data using formulas.
- How to summarize KPIs using charts and cards.
- How to visualize insights in PowerBi.
